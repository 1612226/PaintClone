# Paint-Clone
## Prerequisite
Windows
Visual Studio 2013 or newer
## How to run?
Build the project (with release mode)
## Usage
<ol>
  <li> Draw (Line, Rectangle, Circle, Bezier)</li>
  <li> Fill Color (Support multiple color, pattern)</li>
  <li> Delete </li>
  <li> Choose multiple object </li>
  <li> Save and Import </li>
</ol>
